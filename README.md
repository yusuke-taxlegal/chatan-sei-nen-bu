# 北谷町商工会青年部 部員情報共有システム

## 概要

このプロジェクトは、北谷町商工会青年部の部員同士の交流とビジネスマッチングを促進するための情報共有システムです。

---

## 開発・運用ガイドライン

### 必要なツール
- Node.js (v18以上を推奨)
- yarn

---

### ローカル開発環境のセットアップ

ローカルで開発を行うことで、変更内容をデプロイする前にPC上で確認できます。

1.  **ディレクトリの移動**
    ```shell
    cd chatan-sei-nen-bu
    ```

2.  **依存関係のインストール**
    プロジェクトに必要なライブラリをインストールします。（初回のみ）
    ```shell
    yarn install
    ```

3.  **開発サーバーの起動**
    ```shell
    yarn dev
    ```
    ターミナルに表示されたURL（例: `http://localhost:5173/`）をブラウザで開きます。
    コードを編集すると、変更が自動でブラウザに反映されます。

---

### 本番環境へのデプロイ

本システムはすでに公開・運用されています。安全に開発を進めるため、以下のガイドラインを必ず遵守してください。

1.  **Gitでブランチを作成:**
    Gitを使っている場合は、`main`ブランチから新しい作業ブランチを作成してから作業を開始することを推奨します。
    ```shell
    # (推奨) 例: mainブランチから「feature/〇〇」ブランチを作成
    git checkout -b feature/add-new-page
    ```

2.  **本番環境へのデプロイ:**
    -   開発が完了したら、**必ず`yarn build`コマンドで最新のソースコードをビルド（変換・圧縮）します。**
        ```shell
        # chatan-sei-nen-bu ディレクトリの中で実行
        yarn build
        ```
    -   ビルドが完了したら、一つ上の階層に戻り、Firebaseへデプロイします。
        ```shell
        # 一つ上の階層 (00_Obsidian) に戻る
        cd ..

        # 本番環境へデプロイ
        firebase deploy --only hosting
        ```

## データベースのデータ追加

部員のプロフィール情報をデータベースに追加（または復元）する場合は、以下のスクリプトを使用します。

### サンプルデータの追加・削除

開発やデモ用に、約10人分のサンプルデータを一括で追加・削除できます。

1.  `chatan-sei-nen-bu` ディレクトリに移動します。
    ```shell
    cd chatan-sei-nen-bu
    ```

2.  以下のコマンドを実行します。
    -   **サンプルデータを追加する場合:**
        ```shell
        node add-sample-data.js add
        ```
    -   **サンプルデータを削除する場合:**
        ```shell
        node add-sample-data.js delete
        ```
        *※このコマンドは、IDが `sample-user-` で始まるユーザーのみを削除するため、安全にご利用いただけます。*

### 特定の部員データを手動で追加する場合

1.  `chatan-sei-nen-bu/add-sample-data.js` ファイルなどを参考に、手動でFirebaseコンソールからデータを追加してください。
